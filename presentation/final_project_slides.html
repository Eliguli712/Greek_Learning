DAG-Based Semantic Analysis for Ancient Greek

Why Analyze Ancient Greek Semantics?

  • Classical texts are often studied through translations, commentaries, and lexical notes
  • Researchers may want explicit structure: who did what, to what, and with which modifiers
  • Ancient Greek is morphologically rich, so surface word forms hide useful information
  • A graph representation can expose entities, events, semantic roles, and dependencies

DAG-Based Semantic Analysis for Ancient Greek

Semantic Analysis Problem

Input

  • Raw Ancient Greek text
  • Accents, punctuation, enclitics, and inflected forms
  • Example source: Aristotle, De Sensu et Sensibilibus

Output

  • Normalized lemmas
  • Semantic token types and roles
  • Relations such as AGENT, THEME, and MODIFIER
  • A validated directed acyclic graph
Goal: make every intermediate layer visible, inspectable, comparable, and evaluable.

DAG-Based Semantic Analysis for Ancient Greek

Types of Outputs

  • Phoneme layer: syllables and phoneme sequences
  • Morpheme layer: candidate roots, endings, and features
  • Lexeme layer: lemma normalization and transliteration
  • Semantic layer: type, role, and semantic class
  • Relation layer: extracted semantic edges
  • DAG layer: graph nodes, edges, and validation report

Previous Work / Context

  • Classical language processing often begins with tokenization, lemmatization, and morphology
  • Semantic parsing often uses graph-like structures to represent events and arguments
  • Rule-based systems can be limited, but they are interpretable and easier to inspect
  • This project focuses on a small transparent compiler rather than an opaque end-to-end model

Aristotle Case Study

  • Source: Aristotle, De Sensu et Sensibilibus
  • Processed excerpt: 5 sentences, 205 tokens
  • Known lemmas matched: about 90 tokens, roughly 44%
  • Fallback matching handled forms such as ζῴων, ἀναπνοή, νόσος, and αἴσθησις
  • Extracted relation types included AGENT, THEME, and MODIFIER
  • DAG validation succeeded for all processed sentences
