# Greek_Learning
This project develops a DAG-inspired, high-fidelity semantic analysis framework for Ancient Greek, with Latin included as a structurally parallel extension layer.  The core aim is to preserve semantic stability by making every intermediate layer visible, inspectable, comparable, and evaluable.

See [QUICKSTART.md](QUICKSTART.md) for runnable commands covering the raw toy
pipeline, UD-backed compiler, dependency-free baseline, historical manual gold,
and large-scale UD proxy evaluation paths.
